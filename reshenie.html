Решение уравнений и систем уравнений

Решение уравнений

Одно из основных задач математического анализа - решение уравнений. Решить уравнение означает найти такое значение переменной, которое удовлетворяет уравнению. Существует много различных методов для решения уравнений, вот некоторые из них:

Метод половинного деления:

x = (a + b) / 2

Метод Ньютона:

x = x0 - f(x0) / f'(x0)

Метод секущих:

xn+1 = xn - f(xn) * (xn - xn-1) / (f(xn) - f(xn-1))
